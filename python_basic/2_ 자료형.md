2-1 숫자형
---------------------------------------------------
- 숫자형은 숫자가 갖는 특성을 프로그래밍 언어로 구현한 **데이터 유형**을 말한다.
- 숫자형 데이터는 정수형(int)과 실수형(float)으로 구분된다.
  - 자연수와 정수는 정수형으로 정의하고, 소수점이 있는 숫자는 실수형으로 정의한다.
  - <img width="420" height="161" alt="image" src="https://github.com/user-attachments/assets/c4a0ea4b-735e-4199-a5d8-9c10c180adc0" />
  - 자료형을 확인할 때 type 함수를 이용한다.
  - 정수형+정수형 결과는 정수형이고 정수형+실수형 결과는 실수형이다.

2-2 문자열
---------------------------------------------------
- 문자열은(string)은 따옴표 안에 입력한다.
- 큰 따옴표(" ") 또는 작은 따옴표(' ')안에 알파벳, 한글, 숫자, 공벡, 기호 등을 입력하면 문자열 데이터로 인식한다.
  - <img width="333" height="141" alt="image" src="https://github.com/user-attachments/assets/f5e8abe8-8b7a-42fb-a898-151d339bc2b8" />
  
### 2-2-1 문자열 인덱싱
----------------------------------------------------
- 인덱스(index)는 어떤 배열 안에 **원소가 있는 순서**를 나타낸다.
- 첫 번째 원소의 위치를 인덱스 0으로 하고, 두번째 인덱스는 1이 된다.
    - E a s y 문자열(길이 4), 인덱스 : {E:0, a:1, s:2, y:3}
- 원소를 선택할 때는 대괄호([]) 안에 위치 순서를 나타내는 **인덱스 숫자**를 입력한다.
- 인덱스 배열의 순서를 뒤에서 앞으로 표현할 수 있다.
  - 끝 위치를 나타내는 인덱스는 -1로 정의한다. E a s y, 인덱스 : {E:-4, a:-3, s:-2, y:-1}
  - <img width="270" height="181" alt="image" src="https://github.com/user-attachments/assets/78d46d3f-bd39-4ed6-a2c8-0652303d4b54" />


### 2-2-2 문자열 슬라이싱
----------------------------------------------------
- 슬라이싱(slicing) 기법은 인덱스 범위를 지정하여 여러 개의 원소를 추출하는 방법이다.
  - [시작 인덱스:끝 인덱스] 형식으로 추출하려는 범위를 입력한다.
  - 범위를 지정할 때 시작 인데스는 포함하지만 끝 인덱스는 포함하지 않는다.
  - <img width="337" height="154" alt="image" src="https://github.com/user-attachments/assets/1817fc4f-c328-4990-8648-2af7ebe77bff" />
- ":" 를 시작 인덱스나 끝 인덱스 위치에 넣으면 처음부터나 끝까지를 의마한다.
  - <img width="309" height="194" alt="image" src="https://github.com/user-attachments/assets/4ac9a9e5-94f6-4b5d-babb-c4e1b071ad7e" />

2-3 리스트
-----------------------------------------------------
- 리스트(list)는 여러 개의 데이터를 하나의 모음(집합)으로 다루고 싶을 때 사용하는 대표적인 자료형이다.
  - 여러 개의 원소를 담는 집합 또는 꾸러미라고 생각하면 쉽다.
  - 리스트를 만들기 위해서는 대괄호([]) 안에 여러 개의 원소를 쉼표(,)로 구분하여 입력한다.

- list() 함수에 배열을 입력하지 않으면 원소가 없는 빈 리스트를 생성한다.
  - <img width="298" height="145" alt="image" src="https://github.com/user-attachments/assets/53d7c1f2-6cfa-4b80-8225-e018c0b40ab6" />
  - []와 같이 원소를 추가하지 않고 대괄호만 입력하는 방법으로도 빈 리스트를 만들 수 있다.

### 2-3-4 리스트에 새로운 원소 추가 
-----------------------------------------------------
- 리스트의 맨 뒤에 새로운 원소를 추가할 수 있다.
- 리스트 객체에 도트(.) 연산자를 이용하여 apped 메소드를 적용한다.
  - <img width="315" height="158" alt="image" src="https://github.com/user-attachments/assets/63d88be5-b740-4230-aa19-7a32b1405dfc" />
- 리스트의 맨 뒤가 아닌 원하는 곳에 원소를 추가할 수도 있다
  - insert(index, x)를 사용하고 기존 원소들은 한 칸씩 뒤로 밀린다. (한 번에 하나의 원소만 추가 가능)
  - <img width="329" height="171" alt="image" src="https://github.com/user-attachments/assets/ea078b16-91b7-4645-9749-5889f746dbe8" />
- 여러 원소를 한 번에 추가할 수도 있다.
  - extend() 또는 +를 사용한다. (리스트의 맨끝에 새로운 리스트 추가)
  - <img width="472" height="232" alt="image" src="https://github.com/user-attachments/assets/fe707216-d920-47f7-8915-86e0847feea4" />
- 여러 원소를 원하는 위치에 넣으려면 슬라이싱 대입을 사용한다.
  - 원하는 인덱스 위치 [i:i]를 사용해서 대입한다.

*주의점*
- lst[2] = x : index 2번 원소를 x로 교체(replace)한다.
- lst[2:2] = x : index 2번 위치에 x를 추가(insert)한다.
